# AI Research Assistant (LangChain + Groq + Tavily)

## Overview
This project is an AI-powered research assistant that uses a large language model (Groq) combined with live web search (Tavily) to answer user questions.

## Features
- LLM-powered reasoning (Groq LLaMA 3.3)
- Live web search (Tavily API)
- Agent-based architecture (ReAct framework)
- Clean CLI interface
- Modular project structure

## Tech Stack
- Python
- LangChain
- Groq API
- Tavily Search API

## How to Run
pip install -r requirements.txt
python main.py
